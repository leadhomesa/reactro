# reactro
retro + react = reactro

## Getting Started
```
cp sample.env .env
npm install
```

## Local development with HMR
```
npm start
```

## Production build
- Set NODE_ENV=production to enable the service worker.
```
npm run build
OR
npm run clean-and-build
```
